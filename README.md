
# Cats Help Donation Campaign Landing Page

This is a code bundle for NGO Donation Campaign Landing Page. The original project is available at https://www.figma.com/design/dDaHmeK37Vk8BzYk0PpUnn/NGO-Donation-Campaign-Landing-Page.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Legacy

Old version of the donations page is available at `old_donations_page` folder.
